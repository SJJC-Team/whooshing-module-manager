# Whooshing 服务模块管理器
由于整个 whooshing 网络系统是高度可扩展的，因此需要一个管理模块负责管理所有的其他服务模块。

该模块需要完成：

1. 密钥存储
2. 密钥轮换
3. 数据库、用户数据加密
4. 服务模块间通讯加密
5. 服务状态控制，例如启动，停止，状态监控等等
6. 管理日志



见[主项目](https://github.com/SJJC-Team/whooshing)
